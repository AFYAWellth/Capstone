Interaction Flow!

Pre-Condition: Jessica has an active profile on the Animal Blog Hub, has engaged with posts previously, and is currently logged into his account.

PostModel-Condition: Jessica can effortlessly create a new blog entry about his pet, view comments, and interact with other pet owners on the platform.

User Login/Signup:

  • The user accesses the website and chooses to sign up/log in.
  • Completes the authentication process using email or social media credentials./li>

Navigating the Homepage:

  • Upon successful login, the user is directed to the homepage where they can see a feed of recent posts.
  • Options to filter the feed by categories such as Dogs, Cats, etc., or by trending topics.

Creating a PostModel:

  • User clicks the “New PostModel” button.
  • Enters the content into a post creation tool which includes options for text, images, and videos.
  • If an image is uploaded, the user has the option to generate a meme: A text box appears suggesting humorous captions based on the animal in the image.
  • Submits the post, which then appears on their profile and the communal feed.

Engaging with Content:

  • User browses the feed and interacts with posts through likes, comments, and shares.
  • Follows other users whose content they find engaging.

Profile Customization:

  • User accesses their profile from the navigation bar.
  • Updates their bio, profile picture, and manages posts.

Log Out/Exit:

  • User logs out through the profile menu when done, ensuring their session is securely closed.
