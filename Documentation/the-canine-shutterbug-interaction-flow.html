Interaction Flow for creating a meme on Talking Pets Meme Generator

Precondition

Sam has an account on Talking Pets Meme Generator site and is going to create a meme to show off a dog photo he took today.

Postcondition

Sam easily created a funny meme and added it to a couple social media sites.

Interaction Flow

User clicks on "Create Meme" or similar button Site populates options for a template User selects a template Site populates option to use upload a file User uploads an image Site populates an editing tool bar User customizes the meme with text to top and bottom of image Site populates a preview of the created meme User previews and saves the meme Site offers to share directly to several social media sites User shares to two social media sites Site offers to send a copy of the meme to its meme library User agrees to add meme to library Site shows status that meme has been added to library User clicks to end session Site asks if user is sure they want to exit User clicks yes Site closes
